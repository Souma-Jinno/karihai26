# karihai25

昇圧コンバータ（DC-DCコンバータ）の設計・実装プロジェクト

## 📋 プロジェクト概要

このリポジトリは、**学部3年生の研究室仮配属者向けの実習プログラム**として作成された、昇圧コンバータ（ブーストコンバータ）の設計から実装までを学ぶための教材です。

### 実習の目的
- **LTspiceによる回路シミュレーション**: 昇圧回路の動作原理を理解し、理論値と実測値を比較
- **KiCadによるプリント基板設計**: 回路図からプリント基板への実装方法を習得
- **実回路の製作と動作検証**: 実際に部品を実装し、設計通りに動作するか確認

### 学習内容
- DC-DCコンバータ（昇圧回路）の基本原理
- PWM制御によるスイッチング回路の設計
- MOSFETゲート駆動回路の設計
- LTspiceを用いた回路シミュレーション
- KiCadを用いたプリント基板設計
- 理論計算と実測値の比較・分析

## 🔬 シミュレーション例

### 昇圧回路の設計パラメータ

```
入力電圧（Vin）: 5V
デューティ比（D）: 0.5 (50%)
理論出力電圧: Vout = Vin / (1 - D) = 5V / 0.5 = 10V
スイッチング周波数: 1kHz (演習で100kHzに変更)

主要部品:
- L1: 700µH（学習用）/ 47µH（実部品）
- C1: 1000µF
- R1: 30Ω（負荷抵抗）
- R2: 10Ω（ゲート抵抗）
- R3: 10kΩ（プルダウン抵抗）
```

## 📊 演習課題

チュートリアルでは以下の演習課題を用意しています:

1. **部品値の変更実験**: 実際に購入した部品（47µH、100µH等）を使用したシミュレーション
2. **周波数の計算**: スイッチング周波数を1kHz→100kHzに変更する方法
3. **デューティ比の影響**: デューティ比と出力電圧の理論値・実測値の比較

## 🚀 実習の進め方

### ステップ1: LTspiceによる回路シミュレーション

1. **LTspiceインストール**: [インストールガイド](LTspice/01_LTspiceインストールガイド.md)に従ってセットアップ
2. **昇圧回路のシミュレーション**: [昇圧回路シミュレーション実践](LTspice/02_LTspice昇圧回路シミュレーション実践.md)で以下を学習
   - 昇圧回路の基本構成と動作原理
   - PWM信号の設定とデューティ比の関係
   - ゲート駆動回路の重要性
   - 理論値と実測値の比較・原因分析
3. **演習課題**: 実際の部品値を使ったシミュレーション、周波数変更の計算

### ステップ2: KiCadによるプリント基板設計

1. **KiCadインストール**: [インストールガイド](Kicad/01_KiCadインストールガイド.md)に従ってセットアップ
2. **回路図の作成**: シミュレーションで設計した回路をKiCadで回路図化
3. **フットプリントの割り当て**: 実際の部品に対応するフットプリントを選択
4. **基板レイアウト**: PCB Editorで部品配置と配線を行う

### ステップ3: 実装・動作確認

1. **部品の実装**: プリント基板に部品をはんだ付け
2. **動作テスト**: 実際に電源を投入して昇圧動作を確認
3. **測定と比較**: オシロスコープで波形測定、シミュレーション結果と比較
